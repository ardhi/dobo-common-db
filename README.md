Tools & utility for [Dobo](https://github.com/ardhi/dobo)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install dobo-extra
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```dobo-extra``` in it
